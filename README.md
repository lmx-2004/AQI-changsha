# AQI-changsha 长沙空气质量指数(AQI)预测

基于长沙市日度空气质量与气象数据，构建多模型对比分析框架，预测未来7日日均AQI。

## 项目简介

本研究以长沙市2022年7月16日至2026年6月16日的逐日监测数据（共1426条）为基础，融合污染物浓度与气象特征，系统构建并对比七类预测模型的性能：

- **传统统计模型**：SARIMA、Holt-Winters
- **机器学习模型**：随机森林、XGBoost、LightGBM
- **深度学习模型**：LSTM、LSTM-ARIMA 混合模型

预测目标为未来7日日均AQI，采用样本外滚动验证逻辑。

## 运行环境

- 依赖库见 `requirements.txt`

## 核心结论

| 模型 | RMSE | MAE | R² |
|---|---|---|---|
| **LSTM** | **21.29** | **16.43** | **0.4801** |
| 随机森林 | 21.58 | 16.04 | 0.39 |
| LightGBM | 22.07 | 16.92 | 0.37 |
| XGBoost | 22.56 | 16.92 | 0.34 |
| SARIMA | 23.96 | 19.50 | — |
| Baseline | 33.52 | 23.95 | 0.00 |
| Holt-Winters | 34.95 | 25.71 | — |

- **LSTM 表现最优**：R²=0.4801，得益于门控记忆机制对AQI累积效应与滞后消散特征的捕捉。
- **随机森林稳健性强**：特征重要性分析表明，AQI滞后项是核心驱动因素，PM2.5、气象因子等多变量交互显著。
- **传统统计模型受限**：Holt-Winters 退化为简单指数平滑，SARIMA 多步预测置信区间发散较快，反映长沙AQI具有强非线性与高波动性。

## 特征重要性（XGBoost Top 15）

关键发现：
1. AQI 滞后项（lag）贡献最高（约28%），其次是空气质量等级滞后项（约19%）
2. PM2.5 滞后项位列第三（约8%）
3. 月份（seasonality）、PM10、O3 等历史滞后特征也有一定贡献

## 未来7日预测（2026-06-17 ~ 2026-06-23）

LSTM 预测 AQI 区间为 57.60 ~ 64.30，整体呈先升后降趋势，峰值出现在6月18日。
